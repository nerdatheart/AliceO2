11 /* Singleton class handling/steering single analyses and their lifecycle
12  * 1. registration
13  * -> each analysis is automatically registered
14  * -> the Analysis base class provides registration during construction so there is no
15  * stand-alone analysis
16  * 2. bookkeeping of analyses objects (histograms and meta info)
17  * -> an AnalysisFileHandler does the Bookkeeping of all objects produced during the analyses
18  * 3. initialize
19  * -> steering all registered analyses to initialize their analysis objects
20  * 4. analyze
21  * -> forwarding the steps and magnetic field calls per event to registered analyses
22  * 5. finalize
23  * -> steering all analyses to finalize its objects
24  */
