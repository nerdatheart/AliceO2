Detailed Description

The magic constructors: take arbitrary number of headers and serialize them into the buffer buffer allocated by the specified polymorphic allocator. By default allocation is done using new_delete_resource. In the final stack the first header must be DataHeader. all headers must derive from BaseHeader, in addition also other stacks can be passed to ctor.
