Detailed Description

The class provides a simple interface to the CDB for the TPC specific objects. It will not take ownership of the objects, but will leave this to the CDB itself. This class is used in the simulation and reconstruction as a singleton. For local tests it offers to possibility to return default values. To use this one needs to call 

CDBInterface::instance().setUseDefaults();

at some point. It also allows to specifically load pedestals and noise from file using the 

loadNoiseAndPedestalFromFile(...)

function
